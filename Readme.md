# NLock

NLock makes file security effortless and fast, a reliable lock/unlock built on modern .NET 9 with seamless Windows Explorer context menu integration. One click secures a single file or entire batches, so you stay focused on work, not workflows. Backed by MFA (password + TOTP), only you, with your authenticator app, can access protected content. Lightweight, intuitive, and performance-tuned, NLock fits naturally into your daily routine without a learning curve.

## Why NLock
- Instant lock/unlock for sensitive files
- Simple, clean UI with user-friendly workflows
- Lightweight, performant, and secure by design
- Deep integration with Windows Explorer context menu for one-click lock/unlock
- Batch operations to lock or unlock multiple files at once
- MFA security: protect with password + TOTP (Time-based One-Time Password)

## Get Started
- Download the MSI installer: https://appsdistributionstorage.blob.core.windows.net/msi/Narula/NLock/Narula.File.NLock.Setup.msi
- Run the installer and launch NLock
- Lock or unlock files in seconds - no learning curve
- Right-click files in Explorer to lock/unlock instantly
- Use an authenticator app (e.g., Microsoft Authenticator, Authy, Google Authenticator) to generate TOTP codes when unlocking
